Chapter 2: Protection Strategies

Best Practices to Stay Safe

1. Verify Sender Identity
✅ Safe Practice:
Contact the organization directly using official contact information, not the details provided in the suspicious email.
2. Check URLs Before Clicking
✅ Safe Practice:
Hover over links to see the actual destination. Type URLs directly into your browser instead of clicking email links.
3. Use Two-Factor Authentication
✅ Safe Practice:
Enable 2FA on all important accounts. Even if passwords are compromised, 2FA provides an extra layer of security.
4. Keep Software Updated
✅ Safe Practice:
Install security updates promptly. Use reputable antivirus software and keep it updated.
